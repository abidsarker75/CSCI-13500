## Program Name:

Calculator.cpp

## Purpose:

This program makes a basic calculator with the operations +, /, -, \*, and %.
It will ask users for two numbers and then result the output depending on the
operation. If the numbers or operations are invalid it will tell the user
the inputs/ input are invalid.

## Variables:

| Variable  | Type   | Purpose                          |
| --------- | ------ | -------------------------------- |
| num1      | double | Stores users first number        |
| num2      | double | Stores users second number       |
| operation | char   | Stores letter grade              |
| number1   | int    | Change double into int to modulo |
| number2   | int    | Change double into int to modulo |
